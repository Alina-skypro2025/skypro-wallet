# skypro-wallet
Учебный проект по курсу Skypro.  
Приложение для учёта расходов с возможностью регистрации, входа, добавления расходов и анализа трат по категориям на основе календаря.

---

##  Цели проекта

- Освоить основы командной и индивидуальной разработки.
- Научиться работать с Git и GitHub.
- Научиться работать со спринтами и задачами.
- Сверстать интерфейс по макету Figma (pixel perfect).
- В дальнейшем — реализовать взаимодействие с API и бизнес-логику.

---

##  Страницы приложения

Проект включает следующие экраны:

- Регистрация  
- Вход  
- Мои расходы  
- Форма «Новый расход»  
- Анализ расходов (календарь + графики)
